Object 类是 Java 中所有类的始祖，Java 中的每一个类都扩展了 Object。





1. equals 方法：Object 类中实现的 equals 方法将确定两个对象引用是否相同。
2. getClass 方法将返回一个对象所属的类
3. hashCode 方法：散列码 ( hashcode) 是由对象导出的一个整型值。两个相等的对象要 求返回相同的散列码。
4. toString 方法：它会返回一个字符串，表示这个对象 的值。【绝大多数 (但不是全部) toString方法都遵循这样的格式：首先是类名，随后是一对方括 号括起来的字段值。
